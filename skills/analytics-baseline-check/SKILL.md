# Analytics Baseline Check

## Purpose

Confirm that the site can support basic measurement of enquiries and core interactions.

## Inputs

- Site conversion goals
- Forms and CTA paths
- Existing analytics expectations

## Outputs

- Analytics readiness notes
- Tracking gap list
- Event or measurement recommendations

## Steps

1. Confirm the primary conversion actions that matter.
2. Review whether forms, calls, and other key CTAs are measurable.
3. Note missing baseline tracking decisions or handoff requirements.
4. Keep recommendations proportionate to SMB website scope.

## Done criteria

- Main conversions have a basic measurement plan.
- Gaps are described clearly.
- No unnecessary analytics complexity is introduced.

## Escalation

Escalate when measurement depends on tools, access, or compliance decisions that are not yet available.
