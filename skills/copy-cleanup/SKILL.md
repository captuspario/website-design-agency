# Copy Cleanup

## Purpose

Tighten existing copy so it reads more clearly and credibly while staying faithful to what is actually known.

## Inputs

- Existing copy
- Messaging framework
- Confirmed business facts

## Outputs

- Cleaner page copy
- Revised headings and CTA language
- List of unsupported claims removed or flagged

## Steps

1. Remove filler, repetition, and vague hype.
2. Tighten headings and section intros.
3. Improve scanability with shorter, clearer phrasing.
4. Preserve useful specifics and proof details.
5. Mark any claims that still need confirmation.

## Done criteria

- Copy is plainer and more specific.
- No new unsupported claims are introduced.
- The page intent remains clear.

## Escalation

Escalate when important copy decisions depend on missing facts, proof, or approvals.
