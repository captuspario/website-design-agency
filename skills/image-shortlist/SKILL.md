# Image Shortlist

## Purpose

Choose candidate imagery that supports credibility, clarity, and design direction without auto-locking final assets too early.

## Inputs

- Existing imagery
- Design direction
- Image policy
- Available external sources if needed

## Outputs

- Shortlisted image options
- Source context notes
- Replacement recommendations

## Steps

1. Review current imagery for quality, relevance, and trust impact.
2. Keep strong client-owned images where possible.
3. Shortlist better alternatives when current images are weak.
4. Document source, usage status, and any licensing considerations.

## Done criteria

- The shortlist improves relevance or credibility.
- Source context is recorded.
- Final asset decisions are left for approval where needed.

## Escalation

Escalate when the project lacks usable imagery and external options would materially affect brand trust or legal certainty.
