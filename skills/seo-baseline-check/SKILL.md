# SEO Baseline Check

## Purpose

Confirm that basic on-site SEO hygiene is covered without turning the project into a full SEO campaign.

## Inputs

- Built pages
- Page titles and headings
- Internal link structure

## Outputs

- SEO baseline notes
- Fix list
- Open SEO risks

## Steps

1. Review page titles and meta descriptions.
2. Check heading structure and page intent alignment.
3. Review internal linking opportunities.
4. Confirm basic local search hygiene where relevant.

## Done criteria

- Core pages have usable titles and headings.
- Obvious internal link gaps are identified.
- The output stays within baseline SEO scope.

## Escalation

Escalate when SEO expectations exceed basic site structure and metadata work.
