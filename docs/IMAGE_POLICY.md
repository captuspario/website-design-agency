# Image Policy

## Default preference

- Prefer client-owned imagery when it is accurate and good enough to support trust.
- Use stronger external sources only when current imagery clearly hurts the result.

## External imagery use

- Use free sources carefully when needed.
- Avoid cliché stock when possible.
- Prefer context-rich imagery that matches the service, place, or customer expectation.

## Selection method

- Shortlist options rather than auto-finalising.
- Review image choices at the relevant approval gate.
- Keep image treatment consistent with the design direction.

## Documentation

- Record source context for shortlisted external imagery.
- Note whether an image is client-owned, licensed, free-source, or temporary placeholder.

## Hard limits

- Do not imply ownership or authenticity that is not real.
- Do not use images that weaken trust through obvious generic stock cues.
