# Initial Changelog

This initial template includes:

- a root `COMPANY.md` manifest for Paperclip import
- team definitions for leadership, strategy, design, production, quality, and reporting
- agent role definitions for the full operating model
- tightly scoped reusable skills
- policy and method docs for workflow, approvals, QA, WordPress constraints, research, and client project setup

The package is intentionally limited to simple SMB service website redesign work.
